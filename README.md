# Social Network Graph Analysis

A project to model and analyze a social network using Graph Neural Networks.

## Phases
- Phase 1: Project setup & version control
- Phase 2: Data modelling & feature design
- Phase 3: Matrix computation (A, D, B, L)
- Phase 4: GNN computational graph & node embeddings
- Phase 5: Dynamic frontend visualization
- Phase 6: Integration & GitHub Pages deployment

## Tech Stack
- Python, NetworkX, PyTorch Geometric
- HTML, CSS, JavaScript (D3.js / Vis.js)
- Git & GitHub
